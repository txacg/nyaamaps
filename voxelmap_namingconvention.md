## 字母前缀( [前缀]-[名称] )
* A = 地区
 * AA = 小地区
 * AB = 商业区
* F = 农牧
* H = 住宅，私人用设施
 * H-empty = 未居住
* C = 公共设施（小）
 * C-uncon = 建设中
 * C-unused = 未使用
* O = 建筑或建筑群（大）
* S = 车站
* B = 商店/回收站
* r = 铁路(Polygon Line) 格式：r-hashname-pointNo e.g. r-a-1 r-a-2 r-a-3，会自动转换成多边形。pointNo最小是0，可以不连续。   r-b-1 r-b-2是另一个多边形
 * rs = 步行线路 (Polygon Line) 格式同r

## 多字母前缀( [前缀]-[名称] )
* TELE = 下界传送门/传送点
 * TELE-a-X-b = 串流
* FLAG = 旗子，标识物
* HIDDEN = 可能很难找到的地点

## 符号前缀( [前缀][名称] )
* $ = 值得标记的地方
* & = 坑
* ? = 未确定

## 符号后缀( [名称][后缀] )
* ? = 可能确定

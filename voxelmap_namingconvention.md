## 字母前缀
* A = 地区
 * AA = 小地区
 * AB = 商业区
* F = 农牧
* H = 住宅，私人用设施
 * H-empty = 未居住
* C = 公共设施（小）
 * C-uncon = 建设中
 * C-unused = 未使用
* O = 建筑或建筑群（大）
* S = 车站
* B = 商店/回收站
* r = 铁路(Polygon Line) 格式：r-hashname-pointNo e.g. r-a-1 r-a-2   r-b-1
 * rs = 步行线路 (Polygon Line) 格式：rs-hashname-pointNo e.g. rs-a-1 rs-a-2

## 多字母前缀
* TELE = 下界传送门/传送点
 * TELE-a-X-b = 串流
* FLAG = 旗子，标识物

## 符号前缀
* $ = 值得标记的地方
* & = 坑
* ? = 未确定

## 符号后缀
* ? = 可能确定
